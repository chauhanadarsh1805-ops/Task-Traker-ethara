# Team Task Manager (Full Stack)

A full-stack task management system with role-based access control built using MERN stack.

## Features


- User Authentication (Login/Register)
- Admin & Member Roles
- Task Creation & Assignment
- Task Tracking Dashboard
- MongoDB Database
- Railway Deployment

## Tech Stack

Frontend:
- React
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- JWT Authentication
